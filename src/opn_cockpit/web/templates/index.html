    Tresor-Einstellungen

    Auto-Sperre

    Nach dieser Dauer ohne Interaktion verlangt das Cockpit das Master-Passwort erneut. 1–240 Minuten. Aenderung wird im Tresor selbst gespeichert.
    Gespeichert.

    Backup vor Apply

    Sicherheitsnetz fuer Routen- und Alias-Rollouts. Scheitert das Backup auf einem Geraet, wird auch der Apply auf diesem Geraet uebersprungen (Audit-Eintrag). Backups liegen lokal unter <app_data>/backups/<device_id>/ und sind per Browser-Download wieder zugaenglich.
    Aelteste werden geloescht sobald das Limit ueberschritten ist. Manuelle Downloads teilen sich diesen Pool, damit sie nicht durch haeufige Apply-Backups verdraengt werden.
    Eigener Pool fuer die geplanten Hintergrund-Snapshots.

    Geplante Auto-Backups

    Der Server zieht im konfigurierten Intervall pro Geraet ein verschluesseltes Konfig-Backup. Erscheint auf der OPNsense-Seite als API-Zugriff (eigene Audit-Eintraege dort) — vor Aktivierung mit deinen Compliance-Vorgaben abgleichen. Default AUS.
    1 Stunde bis 7 Tage (168h). Default 24h. Pro Tresor; greift sobald gespeichert (kein Restart noetig). Backups landen im selben gzip-Index wie Pre-Apply-Backups mit trigger="scheduled".
    Gespeichert.

    Achtung: Alle anderen aktiven Sessions werden invalidiert. Andere User muessen sich neu einloggen. Du selbst bleibst eingeloggt, deine Session zeigt danach auf den neuen Tresor.

    Anderen Tresor laden

    Sperrt deine aktuelle Session und oeffnet stattdessen einen anderen oder neu angelegten Tresor. Der Pfad darf auf jedem lokalen Laufwerk liegen (z. B. USB-Stick).

    Tresor exportieren

    Backup-Download

    Laedt die aktive .opnvault-Datei verschluesselt herunter. Inhalt nur mit dem Master-Passwort lesbar. Geeignet fuer Backups (Cloud-Storage, E-Mail an dich selbst, externer Datentraeger).

    Template-Export

    Erzeugt eine Kopie deines Tresors mit allen Firewall-Stammdaten, aber leeren API-Credentials. Zum Weitergeben an andere Admins, die ihre eigenen OPNsense-Keys einsetzen.

    Der Empfaenger braucht dieses Passwort zum Oeffnen. Kann dasselbe wie das aktuelle Master-PW sein, sollte es aber nicht.
